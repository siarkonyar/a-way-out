# A Way Out — CLAUDE.md

## Project Overview

**A Way Out** is an iOS app that lets users physically block distracting apps using any standard, blank NFC tag they already own. It is conceptually similar to [Brick](https://getbrick.com) but removes the hardware purchase barrier: instead of buying a proprietary NFC device, users write to (or simply register) their own NFC tag and use it as the physical key to toggle app blocking on and off.

The core philosophy: your phone should require a deliberate physical action — touching a real-world object — to unblock distracting apps. The NFC tap creates friction that interrupts unconscious phone habits.

---

## Core Differentiator vs. Brick

| Brick | A Way Out |
|---|---|
| Requires purchasing a proprietary NFC device | Works with any blank NFC tag |
| Tag is pre-paired at factory | User pairs their own tag during onboarding |
| One device, one Brick | User controls which tag is assigned |
| No tag reassignment | User can reassign to a new tag at any time |
